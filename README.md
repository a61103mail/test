1. 基礎指標區 (Technical Indicators)
這部分告訴你「現在」發生了什麼事：

相對強弱 (RSI)：衡量買賣力道的動能。數值高於 70 通常代表超買，低於 30 代表超賣。

趨勢均線 (EMA)：代表價格的平均趨勢。價格在均線上方通常看多，下方看空。

動能指標 (MACD)：判斷價格加速或減速的趨勢。

成交均價 (VWAP)：這是一條非常重要的「主力成本線」，價格在 VWAP 之上代表當天買入的人平均是賺錢的。

布林通道 (BOLL)：衡量波動率。顯示「下破」代表價格跌破了支撐軌道，「區間」代表在震盪。

2. 括號內的百分比 (單一指標勝率)
這是這個模型最精華的地方：歷史數據回測。

意思：括號內的數字（例如：36%）代表：「在過去一段時間（通常是 500-1000 根 K 線）中，當這個指標出現『空』訊號時，最後真的賺錢的機率。」

如何解讀：如果勝率低於 50%，代表這個指標在這個幣種上目前「很不準」。這就是為什麼截圖 1 雖然滿眼都是「空」，但總分卻很低，因為歷史告訴你跟著這些指標做空會賠錢。

3. 誘空/誘多風險 (誘空風險 / Bull & Bear Traps)
這解決了「指標全亮紅燈/綠燈卻不准」的問題：

誘空風險：當所有指標都指向「空」，但做空的歷史勝率極低時，系統會判定這是一個「假跌破」或「最後一跌」。

邏輯：因為數據顯示，每次指標變紅時，價格往往會反轉向上。這就是你問的「明明指標看空，為什麼建議做多」的原因。系統正在執行反向操作 (Contrarian Trading) 的邏輯。

4. 多週期矩陣 (15m / 1h / 4h)
這是在判斷長線與短線的關係：

4小時 (趨勢層)：大方向。如果 4H 是綠的，代表大趨勢向上。

1小時 (波段層)：中線節奏。

15分鐘 (執行層)：短線進場點。

共振效應：當三個時區的勝率與方向一致時，系統會給出「強烈建議」。

5. 量能 (Volume) 指數
截圖 5 中的 「量能(1H) x0.6」：

x0.6 的意思：目前的成交量是過去 20 根 K 線平均成交量的 0.6 倍。

判斷：

> 2.0 (爆量)：代表有大資金進入，訊號非常可靠。

0.8 ~ 1.2 (正常)：跟平常差不多。

< 0.6 (縮量)：代表市場沒人玩，這時候的指標變動很容易是「假訊號」或「騙線」。

總結
這個系統不是在「猜」漲跌，而是在告訴你：「歷史上，當出現這種組合時，哪一邊贏面比較大。」
